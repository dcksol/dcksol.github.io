커스텀 패키지 사용하기

Note

manim을 업그레이드 하면 아래 설정이 초기화 되니 주의하자. (같은 커뮤니티 버전이어도 다른 하위 버전간 호환성이 안 좋은 편이다. 또한 초기 버전인 v0.1.0이 2020년 10월 22일 릴리즈된 이후로 거의 매달 새로운 버전이 릴리즈되었다.)

v0.15 이하의 버전에 해당

manim\utils\tex_templates.py을 살펴보면 manim의 기본 preamble로 다음과 같은 템플릿을 사용한다는 것을 알 수 있다.

\usepackage[english]{babel}
\usepackage[utf8]{inputenc}
\usepackage[T1]{fontenc}
\usepackage{lmodern}
\usepackage{amsmath}
\usepackage{amssymb}
\usepackage{dsfont}
\usepackage{setspace}
\usepackage{tipa}
\usepackage{relsize}
\usepackage{textcomp}
\usepackage{mathrsfs}
\usepackage{calligra}
\usepackage{wasysym}
\usepackage{ragged2e}
\usepackage{physics}
\usepackage{xcolor}
\usepackage{microtype}
\DisableLigatures{encoding = *, family = * }
\linespread{1}

만약 위의 패키지 외에 추가로 사용하고 싶은 패키지가 있다면, 공식 문서에서는 다음과 같이 템플릿을 만들어 사용하는 것을 설명한다.

from manim import *

class Sample(Scene):
    def construct(self):
        myTemplate = TexTemplate()
        myTemplate.add_to_preamble(r"\usepackage{kotex}")
        tex = Tex(
            r"Hi 안녕하세요, xyzabc$xyz$}",
            tex_template=myTemplate,
            font_size=72,
        )

만약 일부 특정한 문자에만 특정한 패키지를 사용하고 싶다면 위와 같이 하는 것이 좋으나, 한글 사용을 위한 kotex 패키지와 같이 항상 사용해야할 패키지가 있다면 아예 위의 tex_templates.py에 추가하여 사용하는 것이 좋다. 특히 특정한 프로젝트에서 파이썬을 venv 가상 환경으로 사용한다면, 그 프로젝트에서 사용하는 preamble을 수정해 주는 것이 편리할 것이다.

예를 들어 다음과 같이 쓸 수 있다. (mtpro2 패키지는 대부분의 물리학 교재에서 사용하는 유료 수식 폰트로 별도의 설치가 필요하다.)

_3b1b_preamble = r"""
\usepackage[utf8]{inputenc}
\usepackage{kotex}
\usepackage{amsmath}
\usepackage{amsthm}
\usepackage{setspace}
\usepackage{physics}
\usepackage{newtxtext}
\usepackage[subscriptcorrection,mtpcal,mtphrd,nofontinfo]{mtpro2}
\usepackage[version=4]{mhchem}
\usepackage{physics}
\usepackage{bm}
"""

추가로 manim\utils\tex_file_writing.py의 with open(tex_file), with open(log_file), with open(result) 부분에서 각각 파라미터 encoding="utf-8"을 추가해야 정상적으로 한글이 출력될 수 있다.

v0.17 이후

v0.17 이후는 단순하다. manim\utils\tex.py에서 default_preamble을 수정하기만 하면 된다. 이외의 다른 파일(tex_file_writing.py 등)을 수정할 필요는 없다.
